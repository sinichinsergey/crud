In project used Python 3.10 and PostgreSQL 14.

1. Create venv.
2. Install requirements from "requirements.txt".
3. Create tables in database from "schema.sql".
4. Add data to "config.ini".
5. Fill database by run "seed.py".
6. Start app by run "main.py".
7. Open http://127.0.0.1:8000/docs in browser.
